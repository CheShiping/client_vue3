# 大学答辩管理系统

## 1. 系统概述

大学答辩管理系统是一个基于Vue 3、Element Plus和TypeScript开发的现代化Web应用，用于管理大学毕业论文答辩的全流程，包括答辩计划、答辩分组、答辩评分和答辩记录等功能。

### 1.1 系统特点

- **现代化技术栈**：采用Vue 3、Element Plus、TypeScript等主流前端技术，确保系统的稳定性和可维护性。
- **响应式设计**：适配各种设备屏幕，提供良好的用户体验。
- **模块化架构**：系统采用模块化设计，功能清晰，易于扩展和维护。
- **角色权限管理**：支持管理员、教师和学生三种角色，不同角色具有不同的权限和操作界面。
- **完整的答辩流程**：从答辩计划制定、答辩分组、答辩评分到答辩记录，覆盖答辩全流程。

## 2. 功能模块

### 2.1 用户管理

- **用户登录**：支持用户名密码登录，支持记住密码功能。
- **用户信息管理**：查看和修改个人信息，包括头像、昵称、邮箱、手机号等。
- **密码修改**：修改登录密码。
- **角色权限**：根据不同角色（管理员、教师、学生）显示不同的功能菜单。

### 2.2 学生管理

- **学生列表**：查看所有学生信息，支持分页和搜索功能。
- **学生详情**：查看单个学生的详细信息。
- **学生导入**：支持批量导入学生信息。

### 2.3 教师管理

- **教师列表**：查看所有教师信息，支持分页和搜索功能。
- **教师详情**：查看单个教师的详细信息。
- **教师导入**：支持批量导入教师信息。

### 2.4 答辩计划管理

- **计划列表**：查看所有答辩计划，支持分页和搜索功能。
- **计划详情**：查看单个答辩计划的详细信息。
- **新增计划**：创建新的答辩计划，包括计划名称、答辩类型、开始时间、结束时间等。
- **编辑计划**：修改已有答辩计划的信息。
- **删除计划**：删除不需要的答辩计划。
- **发布计划**：将审核通过的答辩计划发布给学生和教师。

### 2.5 答辩分组管理

- **分组列表**：查看所有答辩分组，支持分页和搜索功能。
- **分组详情**：查看单个答辩分组的详细信息。
- **新增分组**：创建新的答辩分组，包括小组名称、答辩计划、答辩地点、答辩时间、组长等。
- **编辑分组**：修改已有答辩分组的信息。
- **删除分组**：删除不需要的答辩分组。
- **成员管理**：管理答辩小组的教师成员。
- **学生管理**：管理答辩小组的学生成员。

### 2.6 答辩评分管理

- **评分列表**：查看所有学生的答辩评分，支持分页和搜索功能。
- **评分详情**：查看单个学生的答辩评分详情。
- **评分标准**：根据论文质量、答辩表现、回答问题、创新能力等多个维度进行评分。
- **评分录入**：录入学生的答辩评分。
- **评分修改**：修改已有答辩评分的信息。

### 2.7 答辩记录管理

- **记录列表**：查看所有学生的答辩记录，支持分页和搜索功能。
- **记录详情**：查看单个学生的答辩记录详情。
- **编辑记录**：修改已有答辩记录的信息，包括答辩日期、答辩时间、答辩地点、答辩记录、答辩结论等。
- **删除记录**：删除不需要的答辩记录。

### 2.8 通知公告

- **通知列表**：查看所有通知公告，支持分页和搜索功能。
- **通知详情**：查看单个通知公告的详细内容。
- **新增通知**：发布新的通知公告。
- **编辑通知**：修改已有通知公告的内容。
- **删除通知**：删除不需要的通知公告。
- **通知中心**：查看与自己相关的通知公告。

## 3. 技术架构

### 3.1 前端技术栈

- **Vue 3**：使用组合式API，提供更好的TypeScript支持和性能优化。
- **Element Plus**：提供丰富的UI组件，支持国际化。
- **TypeScript**：提供类型检查，提高代码质量和可维护性。
- **Pinia**：Vue 3官方推荐的状态管理工具，替代Vuex。
- **Vue Router**：提供路由管理，支持嵌套路由和路由守卫。
- **Axios**：提供HTTP请求支持，支持请求拦截和响应拦截。
- **SCSS**：提供CSS预处理器支持，提高样式编写效率。

### 3.2 项目结构

```
├── public/                # 静态资源文件
├── src/                   # 源代码目录
│   ├── api/               # API请求定义
│   ├── assets/            # 静态资源
│   ├── components/        # 公共组件
│   ├── router/            # 路由配置
│   ├── stores/            # Pinia状态管理
│   ├── utils/             # 工具函数
│   ├── views/             # 页面组件
│   ├── App.vue            # 根组件
│   └── main.ts            # 入口文件
├── .env.development       # 开发环境配置
├── .env.production        # 生产环境配置
├── tsconfig.json          # TypeScript配置
├── vite.config.ts         # Vite配置
└── package.json           # 项目依赖
```

## 6. 常见问题

### 6.1 登录失败

- 检查用户名和密码是否正确。
- 检查网络连接是否正常。
- 如忘记密码，请联系管理员重置密码。

### 6.2 页面显示异常

- 尝试刷新页面。
- 检查浏览器版本，建议使用Chrome、Firefox、Safari等现代浏览器的最新版本。
- 清除浏览器缓存，重新登录系统。

### 6.3 功能不可用

- 检查用户角色和权限是否正确。
- 检查网络连接是否正常。
- 如问题持续存在，请联系系统管理员。
