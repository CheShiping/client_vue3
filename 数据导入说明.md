## 🚀 导入步骤

### 方法一：使用MySQL命令行（推荐）

1. **打开命令行**，进入项目目录：
```bash
cd D:\Desktop\前端期末项目\client_vue3
```

2. **登录MySQL**：
```bash
mysql -u root -p
```
输入密码：`111111`

3. **执行SQL脚本**：
```sql
source quick_test_data.sql
```

或者使用完整路径：
```sql
source D:/Desktop/前端期末项目/client_vue3/quick_test_data.sql
```

### 方法二：使用MySQL Workbench

1. 打开 MySQL Workbench
2. 连接到数据库（用户名：root，密码：111111）
3. 点击菜单：`File` → `Open SQL Script`
4. 选择 `quick_test_data.sql` 文件
5. 点击工具栏的 ⚡ 图标执行脚本

### 方法三：使用Navicat

1. 打开 Navicat
2. 连接到数据库
3. 右键点击数据库 `defense_management_system`
4. 选择 `Execute SQL File`
5. 选择 `quick_test_data.sql` 文件
6. 点击 `Start` 执行

### 方法四：直接复制粘贴

1. 打开 `quick_test_data.sql` 文件
2. 复制所有内容
3. 在MySQL客户端中粘贴并执行
